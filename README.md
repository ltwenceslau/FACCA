# FACÇÃO NEXUS — Sistema de Gestão Industrial

Sistema web completo para gestão de facções de confecção.  
Roda diretamente no navegador, sem necessidade de servidor ou instalação.

## ✅ Como usar

Acesse pelo GitHub Pages ou abra o `index.html` direto no navegador.

## 📦 Módulos

- **Dashboard** — Visão geral de OPs, financeiro e pendências
- **Cadastros** — Fornecedores, Modelos, Colaboradores, Terceiros, Insumos
- **Produção** — Ordens de Produção com wizard de criação (produto → grade → cores → confirmar)
- **Insumos** — Controle de materiais por OP com bloqueio de produção
- **Estoque** — Controle interno de insumos com entradas e retiradas
- **Financeiro** — Contas, despesas, baixas e vales de fornecedores
- **Colaboradores** — Vales e pagamentos com desconto automático
- **Terceiros** — Envios, retornos e pagamentos de facções externas
- **Acertos** — Acertos quinzenais por OP com abatimento de vales
- **Histórico** — Log de todas as alterações

## 💾 Dados

Os dados são salvos automaticamente no **localStorage** do navegador.  
Não requer banco de dados, login ou conexão com internet.

## 🌐 GitHub Pages

1. Suba este repositório no GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `main` e pasta `/ (root)`
4. Salve — o sistema estará disponível em `https://seu-usuario.github.io/nome-do-repo`
